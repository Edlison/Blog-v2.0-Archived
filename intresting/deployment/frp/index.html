通过frp实现代理

Posted on 12-10-2021 In VPS Views:

Intro

部署frp实现内网穿透, http_proxy, socks5

Deploy

选择对应版本下载 项目地址

Explain

frp实现的本质就是客户端与服务端建立长连接, 从而实现内网穿透的效果.

建立未加密的链接

Server

服务端开启一个端口用于与反向代理的服务器进行连接.

dashboard 是服务端的可视化面板.

可以打开一个dashboard查看端口连接情况.

Client

客户端只要与服务端成功连接, 就可以在服务端上暴露本地的端口.

相当于所有知道端口的终端都可以进行匿名访问.

admin 是客户端的可视化面板, 可以修改配置文件.

这里的[ssh]中括号内的名称可以随意定义, 没有影响, 下文会有影响.

注意: 可视化面板绑定的IP, 0.0.0.0允许通过内网IP访问, 而127.0.0.1只能本机访问.
