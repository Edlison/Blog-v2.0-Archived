方案

本地编译, 将编译好的文件上传git并且文件自动部署到nginx

todo: 后期将原文件上传git通过action自动编译并部署

思路1

本地push到github, 通过webhook发送POST请求到VPS, 执行shell进行clone然后拷贝至nginx

问题: 服务器需要处理http请求

思路2

本地同时push到VPS和github, VPS通过hook执行shell拷贝至nginx

问题: 同时push两个仓库感觉不是很优雅
