Posted on 03-05-2021 Edited on 06-08-2021 Views:

介绍

由于不同特征的量纲可能不一致，数值差值较大，可能会影响到数据分析的结果

MinMaxScaler(归一化)

将数据映射到[0, 1]区间

新数据 = (数值 - 最小值) / (最大值 - 最小值)

Z-Score(标准化)

处理后的数据均值为0，方差为1

新数据 = (数据 - 均值) / 标准差
